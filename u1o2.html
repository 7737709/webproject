2.2) Describe whether the IT tools selected were appropriate E.g. Visual Studio Code, Trello etc.

The IT tools that code nation introduced us to and encouraged us to use were in my opinion, an excellent choice. Using Visual Studio to edit my code files was a great experience. Before the course, I had used Notepad ++ and after using VS code there is no chance that I'm going back to notepad. Moving on from VS code and going to Trello and Slack, although two completely different platforms they both helped me with my communication and maintaining full clarity for the duration of the project.
